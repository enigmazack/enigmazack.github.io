蓝光重编码常用工具

发表于 2020-03-24 更新于 2021-06-02 分类于 压制

系统部分

Windows

推荐Windows 7/10，很多工具没有Linux和Mac OS原生版本的，需要wine或者虚拟机。

MSYS2

MSYS2 installer /
Introduction to MSYS2 /
wiki on SourceForge

推荐msys2主要是为了在Windows上使用bash，提高工作效率，非必须。

网络资源

  • videohelp 国外一个网站，收集整理了各种工具，下文提到的工具基本都可以在这个网站找到
  • doom9 许多工具的开发者都在这个论坛
  • nmm-hd 国内版doom9

视频部分

Avisynth

Avisynth wiki /
Avisynth@videohelp /
Avisynth@sourceforge /
Avisynth MT@doom9 /
Avisynth+@doom9

各种fork版本非常多，主要原因是官方只提供32位程序，且只能使用单线程处理。
Avisynth MT是多线程版本，Avisynth+对应64位版本。
如果只是应付蓝光重编码的日常应用，官方版本足够应付，而且对应的插件容易得到。

推荐: 官方32位稳定版

常用插件:

  • ffms2 用于导入mkv视频
  • f3kdb debanding插件
  • FillMargins 处理坏边
  • BalanceBorders 配合FillMargins使用
  • MaskTools2 被BalanceBorders调用
  • Spline36ResizeMod (包含在PTP插件合集中) 内置的Spline36Resize有bug，故有此mod

插件集合
PTP@mega /
插件列表

AvsPmod avisynth 的 frameserver

x264

x264 官方 /
x264 kMod /
avs4x26x

tMod在国内有一些人气，也不断有人fork，但影响力不及kMod。
官方版本和kMod区别不大，个人认为没有必要纠结用那一种。

avs4x26x是astrataro提供的32位Avisynth对64位x264/x265的pipe通道。
avs4x26x默认调用同目录下x264_64.exe，也可设置参数指定要调用的x264编码器。

推荐: avs4x26x调用64位kMod

mkvtoolnix

mkvtoolnix

提供mkv相关的封装和抽取工具mkvmerge/mkvinfo/mkvextract/mkvpropedit，有整合的GUI。

字幕部分

IdxSubOrc

发布地址(作者老马的博客)

老马制作的调用微软MODI ORC中文字幕软件，需要先将蓝光字幕转转成DVD字幕。
关于MODI，老马提供了一个抽取必要组建的包，用起来非常方便。
相关的使用说明都能在作者的博客中找到。

Subtitle Edit

@videohelp /
官网
